还有什么是靠得住的？

September 13th, 2008 阅读量： # 生活用品 # 饮食 # 维权

最近闹的轰轰烈烈的三鹿奶粉事件，再一次将食品安全问题推到了风口浪尖上。我觉得一些黑心企业和制造商一直以来都在瞒天过海的欺骗消费者，只是事情闹大了，才被揭示出来。在我们的饮食品上下毒手，确实太TM可恨了。


前几天刚看完一篇杂志上报道的“生活中常见的有毒食物”，没想到三鹿这边就东窗事发了。相比之下，我看的那篇报道更是令我震惊不已，文中列出了N多有害食物(见下文)，并且都是在生活中我们完全避免不了要食用的食物。虽然以前对这些事也是早有耳闻，一条两条也就不放在心上了，但是看完一下总结出来的这么多“垃圾”食品，还真让人惊恐不已。

本来不想在这件事上跟风，添油加醋，但我想来想去觉得这是关乎自己生命健康的大事，应该揭露出来让更多人看到，为这些黑心人加一把火。早早把他们烧掉才好。在骂声一片中，也给那些想犯错误还没来得及的人提个醒，千万别做愧对自己良心的事，结果是很严重地。

如果你一直坚持诚实守信，那么我们则拥护你，我们拥护光明正大的民族企业。(下文附有含毒食物一览表，不管是不是危言耸听，今后大家都要注意一些了)

  • 过期大米：含致癌黄曲霉素，发黄至发黑，有微腥味，是不能直接使用的陈化粮。
  • 绿色大米：加入工业染料“绿亮剂”和香精、色素等，假冒竹香米。
  • 馒头：用化工原料硫磺熏面粉，可使面团发得更大、更白。
  • 饺子：用猪肉的弃料做馅，夹杂有猪毛，加上冻烂发黑的白菜。
  • 面粉：加入工业漂白剂，俗称“雕白块”，摄入10克足以致命。
  • 粉丝：用低成本的玉米粉制作，加入工业漂白剂漂白。
  • 肉松：肉源是病死猪只，甚至用双氧水漂白猪肉。
  • 猪肉：有的养殖场用俗称“瘦肉精”的哮喘药来喂猪。
  • 火腿：为防苍蝇和蛀虫，用杀虫剂泡火腿。
  • 鸭子：用沥青退鸭毛。
  • 假鸡蛋：碳酸钙质蛋壳，树脂制蛋黄。另外，因为市场上红心蛋价格比较高，因此养殖户在鸡饲料里加入色素，造假红  心蛋。经常食用会引起严重贫血、白血病、骨髓病变等。
  • 蔬菜：残余过量农药。
  • 豆芽：大量使用增白剂，保鲜粉等化学物，含过量二氧化碳。
  • 糖醋大蒜：为延长保鲜期，用化学分混合漂白粉加工蒜头。
  • 水果：利用激素，例如膨大剂催熟。
  • 泡菜：用违禁的工业盐腌制。
  • 鲜竹笋：使用硫磺和工业盐处理。
  • 腐竹：曾出现有人用工业明胶、工业染料来做腐竹。
  • 木耳：因市场上黑木耳价格比较高，因此用墨汁着色，为增加重量，掺入糖、淀粉、硫酸铜、硫酸镁，甚至水泥。
  • 蘑菇：为使保存期达到两三年以上，有人用亚硫酸钠浸泡蘑菇，令其颜色更鲜艳。长期食用会严重损害内脏，甚至引发癌症。
  • 大闸蟹：曾出现汗砒霜的大闸蟹，专家估计是因为农村多用砒霜毒老鼠，砒霜随雨水流入湖中。
  • 海虾：为了省下买氧气泵的钱，用尿液浸泡或用尿素饲养海虾。
  • 墨鱼仔：甲醛浓度高达每公斤30毫克。
  • 咸鱼：为防苍蝇，喷洒杀虫农药。
  • 鱼翅：用双氧水加工。
  • 奶粉：劣质”大头奶粉“，婴儿服用后头胀如斗，营养不良，导致多名婴儿死亡。
  • 酸奶：有些厂家生产的酸奶微生物多得数也数不清。
  • 火锅：汤底中有石蜡做的凝固剂。
  • 瓜子：用工业盐、滑石粉等加工，含铝量过高。
  • 瓶装水：直接用溪水、自来水灌装，假冒”纯净水“。
  • 潲水油：用喂猪的潲水提炼食用油。
  • 酱油：曾出现用毛发水酿造含铅、砷、黄曲霉素等毒的酱油。
  • 月饼：一些无良商人将往年没有卖完的月饼回收，用旧馅做新饼。

注：以上有毒食物列表，摘自《发现》第124期。
