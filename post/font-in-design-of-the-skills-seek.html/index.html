在设计中字体的应用技巧(一)，寻找字体

September 7th, 2008 阅读量： # 技巧 # Download # design # 素材 # 字体

在平面设计的排版布局中，字体设置是最重要的环节之一。适当的字体设置不仅让人看着舒服，而且还能起到引导读者的作用。字体设计是一门学问，通过我自己的一些实践，我逐步总结了一些经验技巧，在这里与大家分享一下。如果你有其他好的方法，也可以跟我交流一下，希望能与大家共同提高。

关于字体的应用技巧，我准备用系列文章的形式与大家一起研究。目前我规划好的内容有三个部分，在今后的实践中可能还会有新的话题产生，我会逐步添加到这个系列中，当然这是后话了。在这一节中，你能获得的有效信息是：如何快速找到合适的字体。

好了，开始正题。一般情况下我们用Photoshop选择字体的时候，是一路NEXT点击下去预览的。这样一来就比较麻烦，当我们看好某一字体后，如果还想继续翻看有没有更合适的字体，那该怎么办？要是继续翻下去，结果没找到更合适的，再返回去找最开始比较合适的那款字体时，是不是基本已经忘记它长什么样了？找来找去既浪费时间又浪费精力，真麻烦，如果电脑里字体装的稍微多点，那还不把人找疯了…这一点Photoshop就不如Illustratorcs做的人性化，后者在预览字体时是所见即所得的。

那在做Photoshop时有没有什么好方法解决这个问题呢？当然有，为了以后方便快速查找字体，我特地手动做了一张“字体预览表”。有了这个表，你可以一目了然地查看喜欢的字体，然后对应字体名称快速在电脑中找到相应的字体。

这样是不是就省事多了？还不错吧，呵呵。如果对你有帮助的话，那么就下载这个“字体预览表”吧。值得注意的是，这个图中的字体是经典字库的全部字体，如果你的电脑里没有经典字库，那么得先到下面的地址下载字库安装程序。然后你就可以享用它了（图中的字体大小是按我个人平时使用频率设置的）～

下载链接：经典字库大全(191MB)

提示：在中文字库中，种类很多，几个常见的有经典字库、文鼎字库、汉仪字库。总体来说字体都大同小异，所以这张“字体预览表”可以举一反三作为查找不同类型字库字体的一个参考。
